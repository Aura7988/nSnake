Welcome to nSnake

nSnake is a implementation of the classic snake game with textual interface. It's playable at command-line and uses the nCurses C library for graphics.

Currently, it works on GNU/Linux (tested on Debian/Ubuntu/Fedora) and Microsoft Windows (tested on Windows 7).

The purpose of nSnake game was to provide a simple-coded, almost tutorial-like game with C. The source code is so well-commented that it's like it's talking to you.

Features:

  • Simple textual interface!
  • Two gameplay modes: With or without borders!
  • Largely commented source code!
  • 9 Speed Levels!
  • Highscore support!
$ cd nsnake_folder/
$ make
$ make install

# If you want to test the game first
$ make run
